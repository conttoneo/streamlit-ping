To run the app locally

1. Download the code
   ... or clone it (https://github.com/conttoneo/streamlit-ping.git)
  
2. Install Streamlit
	`pip install streamlit`

3. Run app locally
	`streamlit run Home.py`

To watch demo go to
	https://conttoneo-streamlit-ping-home-wytoie.streamlit.app/Check_Status
