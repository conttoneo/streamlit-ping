Lokale Ausführung der App

1. Code herunterladen
   ... oder ihn clonen (https://github.com/conttoneo/streamlit-ping.git)

2. Streamlit-Installieren
    `pip install streamlit`

3. App lokal ausführen
    `streamlit run Home.py`
